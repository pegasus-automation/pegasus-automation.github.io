FAQs

Frequently Asked Questions

Dive into the following questions to gain insights into the powerful features that MYOSA offers and how it can elevate your development journey.

What is MYOSA?

MYOSA (Make Your Own Sensors Applications) is a plug-and-play, multipurpose, multi-sensor system designed for learning and developing sensor-based applications. It consists of modular components that can be easily connected, allowing users to focus on application development without worrying about complex wiring or coding.

What components are included in the MYOSA kit?

The MYOSA kit includes: Motherboard: An Arduino-derived open-source controller. Sensorboards: Various sensors that can be easily plugged into the motherboard. Actuators: Modules like LEDs and micromotors for output. Bluetooth and WiFi connectivity: For data transfer to the MYOSA mobile app. Mobile Application: For real-time monitoring and data visualization. Firmware: Open-source code that can be modified for custom projects.

How do I connect the sensors and actuators?

All components in the MYOSA system are designed as blocks that can be easily plugged into the motherboard. This plug-and-play feature eliminates the need for jumper wires and complicated setups, allowing you to get started right away.

What kind of applications can I create with MYOSA?

MYOSA can be used to create a variety of applications in areas such as smart cities, health monitoring, agriculture, and more. Sample applications include fire detection, environmental monitoring, and smart home automation. Users are encouraged to explore their creativity and develop innovative sensor-based solutions.

Is the MYOSA firmware customizable?

Yes, the firmware for MYOSA is open-source, meaning you can modify and customize the code according to your project needs. You can also attach additional components to the motherboard and adjust the firmware to accommodate them.

How do I access the data from my sensors?

The MYOSA mobile application allows you to access real-time sensor data. The readings displayed on the OLED screen of the MYOSA motherboard are transmitted via Bluetooth to the app, where you can visualize the data through graphs and monitor changes over time.
